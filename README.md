## Proyecto #1: Tic-Tac-Toe/tres en raya en consola

  
### 📄 Descripcion
Es una simulcaion del juego que muchos conocemos como 'gato' usando **x** y **o**, aqui gana aquel que acomplete una secuencia de **x** o **o**

### 🎰 Reglas del juego 
- La maquina juega con las **X** mientras que el usuario con las **O**
- La maquina siempre hace el primer movimiento y coloca en el centro o sea en la casilla numero 5 la **X**
- El tablero es de 3 x 3 y a excepcion del 5 se puede elegir cualquiera de los otros numeros
- La maquina eleige de manera aleatoria las demas posiciones para sus **X**
- El juego termina cuando alguien logra 3 en raya.

### 🛠️ Lo aplicado
  - Python
  - random (randrange) importando ``from random import randrange``
  - Listas bidimensionales (matrices)
  - Ciclos while y recursividad (if)
  - Entrada por consola
 
### 🚀 Como ejecutarlo
1. Tener python 3 intalasdo
2. Descargar el archivo "gato.py"
3. Ejecutar ya sea en IDLE
